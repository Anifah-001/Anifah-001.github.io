## 🚀 Projects

This template is set up to showcase your best work. You can easily customize the project cards in the `index.html` file.

### Project Title One
### **Comparative Analysis of the Impact of Agricultural GDP and Oil Revenue on the Nigerian Economy**

A data-driven analysis of Nigeria's economic performance (2000–2021), comparing the contributions of Agricultural GDP and Oil Revenue to economic growth. The project utilized **Excel, SQL, Python, and Power BI** for data cleaning, analysis, visualization, and dashboard development. Key findings revealed that **Agriculture contributed more consistently and sustainably to economic growth**, while **Oil Revenue exhibited greater volatility due to fluctuations in global oil market conditions**, highlighting the need for economic diversification.


### Project Title Two
### **Investigating Factors Influencing Academic Performance Change (Decline or Improvement): A Learning Analytics Approach**

A learning analytics project that examined the factors influencing changes in academic performance among **191 undergraduate students** from two different universities in Nigeria. The project utilized **Excel, SQL, Python, and Power BI** for data collection, analysis, visualization, and dashboard development. Key findings revealed that **study habits, sleep duration, financial stress, social media usage, and motivation significantly influenced academic performance**, with motivated students and those maintaining balanced study and sleep schedules showing greater academic improvement. The study provides data-driven recommendations for supporting at-risk students and enhancing academic success.

## 💡 How to Use This Template

Creating your own portfolio from this template is a simple, four-step process.

### 1. Create a New Repository from this Template
Click the green **"Use this template"** button at the top of this repository page, then select **"Create a new repository"**.

### 2. Name Your Repository for GitHub Pages
This is the most important step. To turn your portfolio into a live website, you **must** name your new repository in the following format:
`your-username.github.io`
*(Replace `your-username` with your actual GitHub username.)*

### 3. Customize Your Portfolio
Open the `index.html` file and begin personalizing it. The file contains helpful `<!--💡UPDATE: ... -->` comments to guide you on where to change:
- Your name and title
- Social media links
- The "About Me" section
- Your project titles, descriptions, and links
- Your list of skills

### 4. Add Your Files
- Upload your profile photo and project preview images to the `assets/images/` folder.
- Upload your CV (e.g., in PDF format) to the `assets/` folder.
- Create your favicon and add it to the root folder.

Once you commit your changes, your new portfolio website will be live at `https://your-username.github.io` within a few minutes!
